# 🤖 Machine Learning Internship – Codveda Technology

This repository documents my work during the **Machine Learning Internship at Codveda Technology**.
Throughout this internship, I worked on a series of structured tasks designed to build a strong foundation in machine learning — starting from data preprocessing to implementing advanced models.

---

## 📌 Overview

The internship is divided into three levels, each focusing on a different stage of the machine learning workflow:

* **Level 1:** Data preparation and basic algorithms
* **Level 2:** Statistical models and decision-based learning
* **Level 3:** Advanced models and ensemble techniques

Each level includes two hands-on tasks aimed at developing practical understanding.

---

## 🟢 Level 1: Basic

**Focus:** Preparing data and applying simple machine learning models

### 🔹 Task 1: Data Preprocessing

* Cleaned and prepared the dataset for modeling
* Handled missing values
* Applied label encoding
* Performed feature scaling (standardization)

📓 `Task1_Data_Preprocessing.ipynb`

---

### 🔹 Task 2: K-Nearest Neighbors (KNN)

* Implemented KNN classification
* Experimented with different values of *k*
* Evaluated model performance using accuracy

📓 `Task3_KNN_Classifier.ipynb`

---

## 🟡 Level 2: Intermediate

**Focus:** Building and evaluating predictive models

### 🔹 Task 1: Logistic Regression

* Built a classification model using logistic regression

* Interpreted model behavior

📓 `Task1_Logistic_Regression.ipynb`

### 🔹 Task 2: Decision Tree / Clustering Task

## 🔴 Level 3: Advanced
